# SpEL Formatter

The `SpELFormatter` is a named parameter formatter that evaluates Spring Expression Language (SpEL)
expressions against parameter values. It is part of the `message-format-spring` module and is registered
under the name `spel`.

For full documentation including configuration keys, usage examples, auto application, security
considerations and registration options, see the
[SpEL named formatter](../../formatter/parameter/named/spel.md) page.
