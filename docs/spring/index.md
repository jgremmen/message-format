# Spring

The `message-format-spring` module provides Spring Framework integration:

- **[MessageSupportMessageSource](message-source.md)** -- a Spring `MessageSource` backed by
  `MessageSupport`
- **[SpringAsmAnnotationAdopter](adopter.md)** -- classpath scanning for message annotations
  using Spring's bundled ASM
- **[SpEL Formatter](spel-formatter.md)** -- a named parameter formatter that evaluates Spring
  Expression Language expressions

## Module Coordinates

```
de.sayayi.lib:message-format-spring:<version>
```
